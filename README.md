## Get Started in 3 Steps

### 1. Get Your Credentials
- Go to [ainative.studio](https://ainative.studio) and create an account
- Navigate to **Developer Settings** â†’ Create an **API Key**
- Create a **Project** and note your Project ID

### 2. Open Your AI Assistant
- Open your code editor (VS Code, Cursor, or Windsurf)
- Launch your AI assistant (Claude Code, Copilot, or Codeium)

### 3. Paste the Starter Prompt
- Copy the entire contents of [STARTER_PROMPT.md](./STARTER_PROMPT.md)
- Fill in your credentials (API Key, Project ID)
- Paste into your AI assistant and start building!
